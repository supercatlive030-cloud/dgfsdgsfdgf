# Unblocked Games Website

A collection of classic browser-based games that are playable anywhere, including restricted environments like schools and offices.

## Features

✨ **Available Games:**
- 🐍 Snake - Navigate and grow your snake
- 🐦 Flappy Bird - Fly through obstacles
- 🎮 2048 - Merge tiles to reach 2048
- ⭕ Tic Tac Toe - Play against AI
- 🧠 Memory Game - Match pairs and test your memory

## Quick Start

1. Open `index.html` in any web browser
2. Click on a game to start playing
3. Use the back button or home link to return to the menu

## How to Use

- **No installation required** - Just open in a browser
- **No plugins needed** - Pure HTML5 and JavaScript
- **No internet required** (after loading) - All games run locally
- **Responsive design** - Works on desktop and mobile

## Browser Compatibility

Works on all modern browsers:
- Chrome
- Firefox
- Safari
- Edge

## Local Development

To develop locally:
1. Clone or download this repository
2. Open `index.html` in your browser
3. Modify game files as needed

## Game Controls

**Snake:** Arrow keys to move

**Flappy Bird:** Click/Spacebar to flap

**2048:** Arrow keys to move tiles

**Tic Tac Toe:** Click to place your mark

**Memory:** Click cards to find matches
